# NovaTech Financial Analysis FY2024

## 📊 Project Overview
End-to-end financial analysis of NovaTech 
Pvt Ltd (fictional Indian tech company) 
for FY2024 using MySQL, Python, Excel 
and Power BI.

## 🛠️ Tools Used
- **MySQL** → Database setup & 15 SQL queries
- **Python** → Data analysis & ratio calculation
- **Excel** → Professional formatted report
- **Power BI** → 4-page interactive dashboard

## 📈 Key Findings
| Metric | Value | Status |
|--------|-------|--------|
| Revenue Growth | 17.14% | Strong |
| Net Profit Margin | 21.45% | Strong |
| Current Ratio | 6.31x | Excellent |
| Debt to Equity | 0.34x | Safe |
| Cash Growth | 25.06% | Strong |
